VALYRO — GUÍA RÁPIDA
=============================

1. ¿Qué es esto?
----------------
Valyro es una herramienta que analiza anuncios de Wallapop para ayudarte
a fijar precios de compra/venta.

- Se ejecuta en tu propio ordenador.
- Los datos se guardan en local (carpeta "data").
- No envía la información a servidores externos.

2. Cómo arrancar la aplicación
------------------------------
1) Haz doble clic en:

   start_market_analyzer.bat

2) Se abrirá tu navegador en:

   http://127.0.0.1:5000/

3) Desde ese panel podrás:
   - Lanzar scrapes de productos.
   - Ver recomendaciones de precios.
   - Comparar varios productos.
   - Generar informes HTML.

3. Carpetas importantes
-----------------------
- data\        → base de datos (market_analyzer.db)
- plots\       → gráficos generados
- reports\     → informes HTML

4. Requisitos técnicos
----------------------
- Windows 10 o superior.
- Conexión a Internet (para acceder a Wallapop).
- Es recomendable cerrar la aplicación desde la "X" del navegador 
  y luego cerrar el proceso "MarketAnalyzer.exe" si quieres detener el servidor.

5. Aviso importante
-------------------
Esta herramienta no está afiliada a Wallapop.

Eres responsable de respetar los términos de uso de Wallapop y de cualquier otra
plataforma que utilices. Usa la herramienta con sentido común.
